Onboard Computers and the 2024 Software Patch

Each Voyager spacecraft is equipped with three redundant computer systems: the Computer Command System (CCS), which manages commands and spacecraft health; the Flight Data Subsystem (FDS), which formats science and engineering data for transmission; and the Attitude and Articulation Control Subsystem (AACS), which controls pointing and thruster firings. These proprietary computers were built using CMOS and TTL integrated circuits, including Texas Instruments 7400 series chips, with a combined memory capacity of just 32 kilobytes.

In November 2023, Voyager 1 experienced a major computer malfunction when its FDS began transmitting an unreadable, repeating pattern of ones and zeroes. Engineers at the Jet Propulsion Laboratory determined that a single memory chip, which stored about 3% of the FDS software code, had failed. Because the spacecraft was billions of miles away, a hardware replacement was impossible. Instead, the team spent months developing a software patch to relocate the affected code to other functional areas of the FDS memory. The patch was transmitted to Voyager 1 on April 18, 2024, and on April 20, 2024, the spacecraft successfully resumed sending readable telemetry, demonstrating the incredible resourcefulness of the mission team.
